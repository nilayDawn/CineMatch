This project solves that problem using Machine Learning & NLP to recommend movies based on your taste.

## 📌 Overview

This is a content-based movie recommendation system built using the TMDB dataset.
It suggests movies similar to the one you like by analyzing metadata such as:

- Genres
- Cast
- Keywords
- Movie descriptions

The goal is to simulate how platforms like Netflix recommend content 🎯

---

## ⚙️ How It Works

1️⃣ User selects or enters a movie  
2️⃣ The system processes movie metadata using NLP techniques  
3️⃣ It converts text into vectors  
4️⃣ Uses Cosine Similarity to find similar movies  
5️⃣ Displays top recommended movies instantly

## 📊 Dataset

Includes:
- Movie titles
- Cast & crew
- Genres
- Keywords
- Overviews

---

## ⚠️ Limitations

- 🔸 Only content-based filtering (no user history or ratings)
- 🔸 Does not learn from user feedback
- 🔸 Limited dataset compared to real-world systems
- 🔸 Metadata inconsistencies may affect accuracy
- 🔸 Works perfectly on old movies( before 2017), does not contain much of new movies
